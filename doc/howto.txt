How to prepare a new blog entry

+ add a blog entry in a new md file, for example: 2012-10-14-year-on-dart.md
+ add a link to the blog entry in default.html
+ add an entry to ondart.rss
+ open Git Bush
+ cd git/blog/dzenanr.github.com
+ git status
+ git add .
+ git commit -m 'year on dart'
+ git push
