How to prepare a new blog entry

http://kramdown.gettalong.org/quickref.html

+ add a blog entry in a new md file (_posts), for example: 2012-10-14-year-on-dart.md
+ create a folder for images (figures) in img
+ add a link to the blog entry in default.html (_layouts)

+ open Git Bush
+ cd git/blog/dzenanr.github.com
+ git status
+ git add .
+ git commit -m 'year on dart'
+ git push
Rt

+ verify the blog entry
+ if ok, add an entry to ondart.rss (dzenanr.github.com)
+ git add .
+ git commit -m 'year on dart'
+ git push
